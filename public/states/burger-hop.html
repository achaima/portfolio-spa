Burger Hop

Aim:

The aim of this project was to create a simple game using the following:

  • Javascript
  • JQuery
  • HTML5 canvas
  • CSS

Brief Description:

A single player game where the aim is to collect as many burgers as possible within 30 seconds and avoid the cabbages.
